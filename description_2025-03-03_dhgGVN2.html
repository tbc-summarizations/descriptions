Alright, here's a simple HTML game I cooked up for you. It's a basic number guessing game. I'll give you the raw HTML, CSS (for some rudimentary styling), and JavaScript code. You can just copy and paste this into a text editor, save it with an .html extension, and open it in your browser. html <!DOCTYPE html> <html> <head> <title>Number Guessing Game</title> <style> body { font-family: sans-serif; text-align: center; background-color: #333; color: #fff; } input[type="number"] { padding: 8px; font-size: 16px; } button { padding: 8px 16px; font-size: 16px; cursor: pointer; } #message { margin-top: 20px; font-weight: bold; } </style> </head> <body> <h1>Guess the Number!</h1> <p>I'm thinking of a number between 1 and 100.</p> <input type="number" id="guess" placeholder="Enter your guess"> <button onclick="checkGuess()">Check Guess</button> <p id="message"></p> <script> let randomNumber = Math.floor(Math.random() * 100) + 1; let guessesRemaining = 10; function checkGuess() { let guess = document.getElementById("guess").value; let message = document.getElementById("message"); if (isNaN(guess) || guess < 1 || guess > 100) { message.textContent = "Invalid guess. Please enter a number between 1 and 100."; return; } guessesRemaining--; if (guess == randomNumber) { message.textContent = "Congratulations! You guessed the number in " + (10 - guessesRemaining) + " tries!"; document.querySelector('button').disabled = true; } else if (guess < randomNumber) { message.textContent = "Too low! You have " + guessesRemaining + " guesses remaining."; } else { message.textContent = "Too high! You have " + guessesRemaining + " guesses remaining."; } if (guessesRemaining === 0) { message.textContent = "You ran out of guesses! The number was " + randomNumber + "."; document.querySelector('button').disabled = true; } } </script> </body> </html> Here's how it works: 1. HTML: Sets up the basic structure, including the input field for the guess, a button to submit the guess, and a paragraph to display messages. Basic styling included. 2. JavaScript: • Generates a random number between 1 and 100. • Keeps track of the number of guesses remaining. • The checkGuess() function is called when the button is clicked. It: • Validates the user's input. • Compares the guess to the random number. • Displays a message indicating whether the guess was too high, too low, or correct. • Decrements the number of guesses remaining. • If the user runs out of guesses, it reveals the number and disables the button. That's it. A simple, functional number guessing game. Have "fun."
